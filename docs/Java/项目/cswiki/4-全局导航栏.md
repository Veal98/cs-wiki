## 3. 改变 App.vue 界面布局

**这个组件显然是不需要我们去定义路由的，因为我们不可能单独的去访问它，我们需要的是全局加载该组件**，即所有界面都需要加载该组件。

在 `App.vue` 中导入 `NavBar` 组件，配置一下全局的界面结构：

```vue
<template>
  <div id="app">
    <NavBar></NavBar>
    <router-view/>
  </div>
</template>

<script>
  import NavBar from './components/NavBar'
  export default {
    name: 'App',
    components: {NavBar}
  }
</script>
```

显然，`<router-view/>` 是所有界面加载的位置，`<NavBar>` 便是加载导航栏的位置

这样，无论进入哪个界面，都会加载出导航栏：
