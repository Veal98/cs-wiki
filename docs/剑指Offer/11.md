# 11 - 二进制中1的个数

### 题目描述

 输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示

### 具体代码

- **C++**

```cpp
class Solution {
public:
     int  NumberOf1(int n) {
         int count = 0;
         while(n != 0){
             n = n & (n-1);
             count ++;
         }
         return count;
     }
};
```



- **Java**

```cpp
public class Solution {
    public int NumberOf1(int n) {
        int number = 0; // 1的个数
        String s = Integer.toBinaryString(n); //整数转为二进制字符串
        String [] split = s.split(""); //依次切分
        for(int i = 0;i < split.length; i++){
            if(split[i].equals("1"))
                number ++;
        }
        return number;
    }
}
```
